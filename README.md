## 📱 프로젝트 소개
이 프로젝트는 인터렉티브 게임 제작시스템의 Produce 부분을 담당하는 백엔드 서버 로직입니다.
(.env 파일 미포함)


### ⚙️ 개발 환경
- `ubuntu 22.04.2`
- `NestJS`
- `TypeORM`
- `Express`
- `MySQL`


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
